### To start a project you need to:
- start backend server:
  1. `cd WAIdrobe/apka/backend/`
  2. python3.12 -m venv venv (create virtual environment)
  3. source venv/bin/activate (open virtual environment)
  4. copy paste `requirements.txt` to the terminal (to install necessary libraries)
  5. python3.12 main.py (to start the backend server)
 
- start frontend server:
  1. `cd WAIdrobe/apka/react_native/`
  2. in terminal type `ifconfig`, find `en0` (you will find your ip there after inet: <ip>)
  3. paste your ip to the ip.tsx (in 'constants/' directory)
  4. `npm start`
 
- on the phone:
  1. Install Expo Go
  2. Scan the QR code that's displayed on the terminal
  3. Test it
